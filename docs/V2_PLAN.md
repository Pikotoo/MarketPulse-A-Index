# MarketPulse v2.0 产品方案

> 最后更新: 2026-07-17

---

## 一、产品定位

| 维度 | 答案 |
|------|------|
| **一句话** | A股投资者的每日数据终端——打开就知道今天市场怎么样 |
| **核心原则** | 帮你看清市场真实面目。不预测涨跌，只呈现数据背后的真相 |
| **差异化** | B: 一站式全维度 + C: 算法透明可审计 |
| **对标** | 韭圈儿恐贪指数（单维度）→ 我们做一站式 |
| **目标用户** | 个人投资者（日均看盘5-10分钟） |
| **使用场景** | 每天早盘前打开看一眼，盘中偶尔刷，收盘后复盘 |

---

## 二、合规边界

```
✅ 展示: 0-100 情绪分、分位数、比值
❌ 禁止: 原始成交额、具体金额、个股名称、买卖建议
✅ 新闻: 只展示标题+来源，不爬正文
✅ 日历: 公开日程信息
✅ AI: 只解释数据含义，不荐股
❌ 龙虎榜: 已从方案移除（涉及个股）
```

---

## 三、信息架构（4个Tab）

```
Tab 1: 首页概览
├── 恐贪指数巨型仪表盘（0-100，纯描述）
├── PE 温度计 + 历史分位
├── 北向资金情绪分
├── 融资融券情绪分
├── 两市量能分 vs 5日均值
├── 涨跌比情绪分
├── 限售解禁压力分（未来一周）
├── 板块资金流向Top5
├── 今日宏观日历
├── 今日市场要闻
└── 更新时间戳

Tab 2: 行业扫描
├── 行业热力图（32行业60日动量）
├── 最强Top5 / 最弱Top5
├── 行业宽度变化趋势
├── 防御/进攻比
└── 涨跌比趋势

Tab 3: 深度分析
├── 历史恐贪指数曲线（可拖拽，7D/1M/3M/6M/1Y）
├── 历史相似期对比 — 纯统计陈列
├── 指标对比器（双指标历史叠加）
├── 估值计算器（PE→分位）
├── 宏观六维分解

Tab 4: 数据接入
├── API 文档
├── Key 管理
├── Python SDK 示例
├── 嵌入代码
└── 每日分享卡片生成
```

## 五、数据管线

```
唯一数据源: H:\数据大全\宏观数据\*.day

akshare 下载 → H:\数据大全\宏观数据\  (唯一写入位置)
                    │
                    ▼
           MarketPulse 信号引擎 (只读)
                    │
                    ▼
               0-100 情绪分 → API 暴露
```

**新增数据:**

## 六、推广策略

### 渠道: 抖音

### 定位: AI 技术分享（非荐股）

### 叙事框架
```
"我用 AI 做了一个 A 股数据分析工具
 每天自动抓取市场数据，生成一份情绪报告
 不推荐股票，不预测涨跌，只呈现事实
 有兴趣的来看看"
```

### 内容形式

| 形式 | 时长 | 频次 |
|------|:--:|:--:|
| 工具展示 | 15-30s | 每周2-3条 |
| 情绪日报打卡 | 10-15s | 每天1条 |
| 开发拆解 | 30-60s | 每月1条 |

### 爆款公式
```
黄金3秒: 「我用AI做了一个工具」
中间:    仪表盘画面，数字跳动
结尾:    「每天自动更新，想看今天的扣1」
标签:    #AI工具 #Python #量化 #数据可视化
```

### 转化路径
```
抖音视频 → 评论区置顶 → 主页链接 → MarketPulse 仪表盘 → 免费看 → 注册/付费
```
